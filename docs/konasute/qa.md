    家用版购买与PASELI的充值需要全程科学上网并使用日本节点

    如果你使用的加速器有家用版购买相关的加速也可以使用

    全文将默认你已经科学上网

!!! info "为什么要玩家用版"

    家用版为街机移植到家用PC的版本，等同于正版，受官方认可

    软件有单独对家用电脑进行适配，不会出现HDD那样各种的问题，也支持120帧

    同时Konami也会不定期的在家用版上开放各种活动，例如抽epass卡、抽周边，不过实体礼品寄送仅支持日本地址

!!! info "数据互通"

    成绩、领航员互通，成绩会取街机或是家用的最高值，领航员需要在街机版上抽，家用版会自动同步，除非是街机独占角色
    
    VF、段位并不互通，走自己的

!!! info "价格"

    每月1628日元，折合人民币82左右

    可使用PASELI或者信用卡付款，不支援中国大陆的支付方式，需要使用VISA，MasterCard，JCB

!!! info "什么是PASELI？"

    PASELI是一个Konami面向街机市场推出的电子货币/电子钱包，用来代替传统的投币模式。使得更加多样化的增值服务的诞生（更好的骗米了）。其中大部分的收入归Konami所有，小部分则会分成给机厅，除了供给游戏外，其还可以在机厅内部做货币使用，其为1:1等值日元

!!! info "歌曲与街机版有什么区别"

    家用版采用本体月定制+曲包的模式，截止本文家用大概有900多首曲目是订阅本体后即可游玩的（包含Blaster Gate解锁的），剩下的需要购买曲包，价格根据曲子数量来定

!!! info "游戏有哪些模式"

    游戏只有Normal跟Skill Analyzer，Normal等同于街机的Premium Time，也没有计时器
